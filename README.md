# LLM_local

Petit dépôt contenant des scripts et des données liés aux prédictions de modèles LLM (Mistral, Qwen, etc.).

Structure principale (extrait) :
- `clean/` : scripts et pages de comparaison
- `dashboard_ref/`, `dashboard_max/` : fichiers pour visualisations
- `data_predicition/` : sorties de prédiction harmonisées
- `data_training/` : jeux de données d'entraînement
- `Mistral7B-instruct/`, `Qwen1.5B-instruct/`, `Qwen7B-instruct/` : scripts et résultats par modèle
- `utils/` : utilitaires (F1, ROUGE, harmonisation, etc.)

Usage rapide :
1. Ce dépôt sert principalement de stockage et d'analyse local. Ne pas inclure de données sensibles.
2. Pour exécuter un script python :
   - Assurez-vous d'avoir un environnement Python adapté et les dépendances nécessaires.
   - Lancez : `python script.py` depuis le dossier contenant le script.

Contribuer :
- Faire une branche pour les modifications majeures.
- Vérifier les fichiers volumineux (logs, sorties) avant de les committer.
